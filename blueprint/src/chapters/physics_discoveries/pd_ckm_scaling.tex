\section{CKM Scaling Law}

The CKM matrix elements follow a systematic $\varphi^{-3|i-j|}$ scaling
based on flavor gap.

\begin{definition}[CKM Scaling]
\label{def:ckm_scaling}
\lean{FUST.PMNSMixingAngles.ckm_scaling}
\leanok
For flavor difference $n$:
\[
|V_{ij}| \propto \varphi^{-3n} \quad \text{where } n = |i - j|
\]
\end{definition}

\section{Physical Interpretation}

The factor of 3 in the exponent $\varphi^{-3n}$ relates to:
\begin{itemize}
\item 3 quark flavors ($= \dim\ker(F_\zeta)$)
\item $N_3$ symmetry in the kernel hierarchy
\item The triplet structure of $\ker(F_\zeta)$
\end{itemize}

The scaling law suggests that flavor mixing amplitudes decay exponentially
with flavor gap, with the golden ratio determining the decay rate.
